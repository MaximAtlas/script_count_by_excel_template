Запуск скрипта:
1) скачать php с официального сайта, не младше 8.0, 
2) Добавить в переменную окружения PATH (свойство системы - дополнительно - переменные среды) папку с php.exe,
 для того чтобы php -v из консоли виделось в любой точке системы (после перезагрузки)
3) Скачать composer с официального сайта, перезагрузить ПК
4) Далее работать из консоли в папке расположения проекта. Скачать composer зависимости (через composer install|update),
при необходимости раскоментировать модули php.ini в папке php.
5) открыть localhost:8000 порт командой php -S localhost:8000
6) заполнить шаблон, сохранить
7) Перейти через браузер на страницу localhost:8000 
8) При несостыковках, файл удалить, и обновить страницу (создать заного)
